# SMC-Simulator
instruction level mips simulator

## Architecture
- 8 Registers, 32-bit
- word addresses
- memory available size is 65536 words
- bit 0 is LSB

## Register
- $0 constant 0
- $1 n input to function
- $2 r input to function
- $3 return value of function
- $4 local variable for function
- $5 stack pointer
- $6 temporary value
- $7 return address
